Features of CalyxOS

We're putting the power back into your hands.

Verified Boot

Verified Boot to prevent tampering with the phone's operating system, using cryptographic signatures.

Security & Privacy Built-In

CalyxOS includes Signal for safe communications, Tor for anonymous web browsing, DuckDuckGo for private search and to block trackers advertising, and other free and open source software to keep you safe.

Android Without Spyware

Download apps found on the Google Play Store anonymously using Aurora, and from the F-droid app store.

Regular Updates

CalyxOS receives the latest security updates for Android, fixing software vulnerabilities and security flaws that can put your phone at risk.
